# ProspectAI v0.1.0-rc1

Data: 25 de junho de 2026

## Funcionalidades disponíveis

- busca de empresas por CEP, categoria e raio de até 50 km;
- resolução de endereço pelo ViaCEP;
- geocoding da origem;
- consulta à Google Places API (New);
- confirmação de distância por Haversine;
- interface local em Streamlit;
- visualização tabular dos leads;
- exportação CSV compatível com Excel no Windows;
- exportação Excel `.xlsx`;
- launcher único para API e interface;
- configuração guiada da chave Google no primeiro uso.

## Como usar

1. Instale Python 3.11 ou superior.
2. No Windows, dê duplo clique em `ProspectAI.bat`.
3. Na primeira execução, informe uma chave Google Maps Platform com as APIs
   Places e Geocoding habilitadas.
4. Informe CEP, categoria e raio e selecione `Pesquisar`.
5. Quando houver resultados, use `Baixar CSV` ou `Baixar Excel`.
6. Feche o terminal ou pressione `Ctrl+C` para encerrar os processos.

Também é possível executar pelo terminal:

```powershell
.\scripts\setup.ps1
.\.venv\Scripts\python.exe -m launcher
```

## Dependências principais

- Python 3.11 ou superior;
- FastAPI;
- HTTPX;
- Pydantic Settings;
- Streamlit;
- OpenPyXL;
- Google Maps Platform com faturamento e APIs compatíveis habilitados.

## Limitações conhecidas

- exige conexão com a internet e credenciais válidas para as APIs Google;
- não possui paginação, retry, cache ou persistência;
- não possui autenticação, banco de dados, CRM ou recursos de IA;
- resultados sem coordenadas não podem ter o raio confirmado e são
  descartados;
- a quantidade de resultados depende do retorno único da Places API;
- a interface foi preparada para execução local, não para hospedagem pública;
- a suíte emite um aviso de depreciação do `TestClient` do FastAPI/Starlette,
  sem impacto nos testes ou na aplicação;
- o smoke test desta RC validou processos, interface, busca simulada e
  exportadores. A busca externa real não foi executada porque o ambiente de
  release não possuía uma chave Google configurada.
